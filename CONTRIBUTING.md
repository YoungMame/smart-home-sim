## Steps

1. Create an issue describing the bug or feature request.
2. Create a new branch from the main branch.
3. Make your changes and commit them following the commit message guidelines.
4. Submit a pull request.

## Commit Message Guidelines
- Use the format: `type(scope): description`
- Types: feat, fix, docs, style, refactor, test, chore
- Scope: optional, can be a specific module or component
- Description: a brief summary of the change

example: fix(api): correct device state update logic
